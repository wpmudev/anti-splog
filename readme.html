What do you get in your zip file?

  1. In the docs folder in here you will find the changelog, install instructions, license and support file. If your theme / plugin comes with a manual you will find this here also as a PDF
  2. Under the plugin folder you will find your plugin
  3. You may also have a mu-plugins folder please read the install in that folder if you do
  4. You will also have an install and plugin for our update notifications system please read that for more information
  5. This readme.html file is supplied also as your overall guide

Installing a plugin

  1. Unzip the downloaded zip file.
  2. Please follow the plugin installation instructions in install.txt and upload path.
